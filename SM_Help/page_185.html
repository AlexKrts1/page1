Производство/переработка

    Группа отчетов по актам производства для выборки использует только параметры "С даты/По дату".
    В главном меню доступны следующие отчеты:
    Отчет по произведенной продукции за период - выводит список сгруппированный по количеству произведенных товаров/блюд с закупочными и розничными суммами за указанный период.
    Отчет по использованным ингредиентам - аналогично предыдущего отчета, но выводит список ингредиентов.
    Отчет по перерасходу ингредиентов за период - сравнивает количество запланированного, согласно калькуляционной карточки, брутто расхода ингредиента, с фактически указанным в акте производства. Те ингредиенты которые указано использовать в большем количестве выводятся на печать.
