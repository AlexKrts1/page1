Остатки товарных групп

    В данном подразделе будут рассмотрена группа готовых отчетов по партионным остаткам товаров выделенная красным прямоугольником, на рисунке ниже:
    1. Отчет по остаткам товарных групп по всем складам, кроме Списания. Формирует отчет по суммарным остаткам всех товарных групп, находящимся на всех складах, кроме склада "№3 Списание". Из опций с главной формы используются "Не показывать скрытые товары", "В отчете по остаткам товарных групп сортировка по ОПС/РПС" (если установлено, то означает сортировку по нумерации отчетности товарных групп для предприятий системы потребительской кооперации (значение проставляется в карточке товарной группы соответствующего справочника - см.раздел справки "Справочники/Товарные группы")). В случае большого количества товарных групп занимает сравнительно много страниц, например, если 1000 товарных групп, то отчет займет 20 страниц.
    2. Отчет по остаткам товарных групп по выбранному складу. Отчет по своему виду аналогичен предыдущему отчету №1, опции задействуются точно такие же, но отбор остатков идет только по выбранному на основной форме отчета складу.
    3. Отчет по остатка товарных групп по отделам. Формат вывода отчета аналогичен предыдущим, но есть и отличия: во-первых никакие опции не используются; во-вторых остаток подсчитывается по всем складам, кроме склада "№3 Списание". Группировка идет по отделам, итоги тоже подсчитываются вначале по отделам, затем, в конце отчета, по всем отделам "Итого". Пример части отчета показан на рисунке ниже:
    4. Остатки с последними ценами по всем группам. Интересный отчет, опции отбора не используются, из параметров используется только склад, выбранный на основной форме отчета. Кроме суммарного остатка по каждому товару, сгруппированному по товарным группам, отбражается закупочная и розничная цены последней партии данного товара.
