Переоценка построчно

    Переоценка товаров построчно позволяет создать переоценку выборочно, объединив товары из разных товарных групп, отделов и т.д., а также назначить индивидуальную наценку/розничную цену товарам отличную от таковых остальной товарной группы/отдела.
    Добавлять товары можно по одному или сразу выбрав целую товарную группу или отдел, объединяющий несколько групп.
    Также, можно добавить товар либо одной торговой марки (двойной щелчок в открывшемся справочнике ТМ), либо сделать множественный выбор торговых марок в соответствующем справочнике (см.справку о справочнике торговых марок).
       У выбранных для построчной переоценки товаров отобранный цены и наценки показаны по данным последней партии, но при выполнении переоценки каждая партия будет обрабатываться индивидуально - и это правильно, т.к. каждая партия может иметь свою закупочную цену. 
       Напоминаем, что при отборе партий по каждому товару имеют влияние опции настроек на вкладках "Наценка-Переучет" и "Партии". Например, переоценять или нет отрицательные и нулевые партии или проводить переоценку только по партиям с таким типом валюты как у последней отобранной партии.
    Когда необходимые товары добавлены, посредством нажатия * (звездочка) входите в режим назначения новой розничной цены. Появится следующее диалоговое окно:
     
    Нажимая "ОК для текущего товара" указанные вами в соответствующих окошках значения наценки/розничной цены будут назначены только текущему товару.
    Нажатие же кнопок с правой стороны формы приведет к тому, что данные значения будут присвоены всем товарам в переоценке, причем в случае новой наценки или донаценки или переоценки по валютному курсу новая розничная цена будет вычислена индивидуально для каждого товара.
     Флажек "Округлять к 0.25 / 0.50 / 0.75" действует для всех типов назначения наценки/цены данной формы, не только для случая перенаценк всех товаров списка по фиксированной новой розничной цене. 
    -
    После назначения для каждого товара новой наценки/розничной цены колонка новой наценки станет зеленого цвета. Можно нажимать в меню "Завершить и перейти к переоценке..."
    После сообщения об успешном проведении переоценки, при необходимости, можно воспользоваться пунктом меню "Напечатать ценники".
