Журнал возвратов товара

    Данный журнал необходим для просмотра возвращенных по чекам товаров.
    Для просмотра чека, из которого был возвращен товар покупателю кликните дважды мышкой на интересующей вас позиции.
    Доступен фильтр журнала, в котором можно задать критерии отбора нужных возвратов.
    Значение опций понятны из их названий.
      Печать списка выводит на предварительный просмотр простой список с итогом по отобранным в журнале записям.
