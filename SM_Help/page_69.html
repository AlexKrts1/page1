Изменить товары курсом/валютой


    Данные возможности позволяют установить признак валюты расчета и типа курса валюты в карточке товара по данным последней партии данных товаров. Первый пункт меню делает это только для выделенного контрагента, а второй - для всех контрагентов сразу.
