Справочники

    Справка по меню Справочники.
    В подразделах детально описаны все меню каждого справочника.
    Однако, есть совпадающие по значению меню во всех справочниках, например:
    Обновить - обновляет записи, перечитывая записи с сервера,
    Сортировать по... - обновляет записи, сортируя их по заданному в этом меню поле,
    Печать списка - выводит на печать список элементов справочника (для справочника товаров доступны дополнительные виды отчетов),  
    Экспорт в текст - экспортирует элементы справочника в текстовый файл, сохраняемый в одном каталоге с программой,
    Удалить - удаляет выделенный элемент справочника (некоторые записи невозможно удалить вообще, например, встроенные склады, также перед удалением проверяется участие удаляемого элемента в документах для сохранения целостности данных),
    Закрыть окно - закрыть справочник (некоторые справочники закрываются по нажатию горячей клавиши Esc).
