Остатки по всем складам

    В данном подразделе будут рассмотрена группа готовых отчетов по партионным остаткам товаров выделенная синим прямоугольником, на рисунке ниже:
    1. Общие остатки по всем внутренним контрагентам и товарным группам по всем складам. Формирует отчет по суммарным остаткам всех товарных групп, находящимся на всех складах, кроме склада "№3 Списание", но обратите внимание, что выводятся только группы у которых в карточке товарной группы проставлено значение "Основной внутренний контрагент". Из опций с главной формы используется только "Не показывать скрытые товары". Группировка и суммирование вначале по внутреннему контрагенту, затем "Итого" по всему отчету.
    2. Общие остатки по выбранному внутреннему контрагенту и товарным группам по всем складам. Отчет аналогичен предыдущему, за исключением того, что признаком дополнительного отбора явлется выбранный на основной форме отчета контрагент, который трактуется как внутренний контрагент, указанный в карточке товарной группы как "Основной внутренний контрагент". Из опций с главной формы используется только "Не показывать скрытые товары". Пример на рисунке ниже:
    3. Остатки по выбранному внутреннему контрагенту и товарным группам развернуто по всем складам. По своей логике, отчет идентичен предыдущим, но создает дополнительную детализацию, разворачивая товар внутри каждой товарной группы. Признаком отбора явлется выбранный на основной форме отчета контрагент, который трактуется как внутренний контрагент, указанный в карточке товарной группы как "Основной внутренний контрагент". Из опций с главной формы используется только "Не показывать скрытые товары".  
     Внимание! опция "Не показывать партии с остатком=0" должна быть снята.
