Заголовок расходной накладной

    Заголовок внутреннего перемещения состоит из двух вкладок: обязательной для заполнения вкладки "Общие" и необязательной "Дополнительно / Опт".
    Рассмотрим заполнение первой вкладки "Общие":
        Номер - присваивается автоматически.
        Дата - указывается текущее время проведения операции.
        Отправитель - выберите склад, с которого товар должен отгружаться.
        Получатель - выберите Контрагента - получателя товара.
        Внутренний контрагент - укажите внутреннее предприятие из справочника контрагентов, которое осуществляет операцию (если выбрано в настройках).
        Примечание - укажите при необходимости.
        Доверенность - укажите при необходимости.
        Пользователь автоматически присваивается по вошедшему в программу.
        Суммы подсчитываются автоматически.
        Внутренний, внешний учет - признак разделения накладных для определенных отчетов.
      На второй вкладке "Дополнительно / Опт" значения выбираемых параметров понятны их их наименований.
      Будучи указанными, они могут использоваться в различных выборках из базы данных.
