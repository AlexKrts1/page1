Найти товар...

    Данная возможность служит для поиска товара по названию. В появляющейся желтой строке поиска наберите часть наименования товара, причем поиск нечувствителен к регистру - можно вводить как большими так и маленькими буквами, поиск также ищет введенное сочетание символов и в любых местах наименований, т.е. не обязательно вводить слова с начала, можно вводить середину или конец слов. Старайтесь вводить сочетание символов которое обязательно будет учавствовать в слове, а не целое слово, т.к. оператор, вносящий в базу данных названия товаров, мог его сократить. Можно вводить граммовку , но не указывайте единицу измерения, например "55г" не будет найдено если оператор ввел при приходе "55 г" и т.п. После набора символов, для поиска нажмите Enter, при этом отберутся все записи удовлетворяющие условию поиска и желтая строка поиска исчезнет. Для того, чтобы закрыть ее вручную, без поиска, нажмите Esc.
    Поиск также можно осуществлять по разным кускам строки названия товара, для разделения используйте символ процента %, на картинке выше показан результат такого поиска, т.е. отбираются строки, в которых есть участки текста разделенные символом процента.
