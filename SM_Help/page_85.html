Сотрудники


    Данный справочник позволяет создавать и хранить информацию о сотрудниках, используемую для взаиморасчетов с ними (более подробно смотрите в соответствующем разделе справки "Взаиморасчеты с сотрудниками").
    Коротко опишем заполняемые поля в карточке сотрудника:
    ФИО, дата приема/увольнения, примечание - значение понятно из названия опций.
    Ставка, грн - месячный оклад сотрудника.
    Процент, % - какой процент от выручки будет начислятся сотруднику, при соответствующем типе оплаты.
    Тип оплаты труда - есть 3 варианта (ставка, ставка+%, только %).
    Тип бартера - есть 2 варианта (по рознице или по закупке) - определяет какая сумма будет вычитаться из выдаваемой на руки. Соответственно сотруднику может быть разрешено брать товары магазина по рознице или по закупке в счет зарплаты.
    Тип сотрудника - есть 4 варианта (постоянный, временный, внештатный, уволен) - для фильтров в соответствующих меню и формах.
    Время рабочей смены - по умолчанию параметр равен 8 часам. Это значение используется для вычисления реальной ставки при расчете зарплаты по фактически отработанному времени.
    Дисконтная карточка - карта присвоенная сотруднику, по которой он будет брать в счет бартера товары магазина.
