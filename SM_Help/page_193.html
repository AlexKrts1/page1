Вкладка Отчеты

    Первая вкладка дизайнера используется для отображения ранее сохраненных или предустановленных отчетов, по умолчанию находящихся в каталоге с программой, например, C:\SuperMarket\Reports, можно переуказать каталог.
    Панель инструментов обеспечивает необходимый сервис для этих файлов: открыть (также работает двойной клик на имени запроса в списке), обновить список, удалить запрос.
