Весы с печатью


    Разрешить работу с печатающими весами - активация возможности.

    Склад для программирования весов: - склад для выбора присутствующих на нем весовых товаров и цен на них, выбираемый по умолчанию. При непосредственной работе с весами (см. раздел "Оборудование: Весы с печатью") Вы всегда можете изменить это значение.

    Префикс-признак весового товара - 2 цифры, печатаемые в начале штрихкода на весовой этикетке, при чтении которых сканером на кассе компьютер понимает, что это весовой товар и соответствующим образом обрабатывает оставшуюся часть штрихкода. По умолчанию установлено в "99", рекомендуемые значения - цифры не занятые международной ассоциацией EAN для идентификации стран-производителей. Сама ассоциация выделяет некоторые номера для этих нужд (т.н. коды ограниченной циркуляции), например, "22", однако в силу ряда причин их не всегда удобно использовать.

    Максимальное количество символов в названии товара -до какого числа символов будет усекаться название товара при их программировании в весы (количество символов в весах ограничено размерами этикетки и тех.характеристиками весов).

    Разница в днях между текущей датой и датой, печатаемой на этикетке в режиме упаковщика. Некоторые весы с печатью, в частности SM-300, обладают возможностью автоматического режима фасовки одинакового товара, при этом фасовать можно следующим или другим днем, например, если фасовка происходит ночью. За более подробной информацией обращайтесь в Ваш сервисный центр.

    Сортировать по товарным группам - при выводе на печать отчета по запрограммированным в весы товарам (см. раздел "Оборудование: Весы с печатью"), отчет будет формироваться с группировкой по товарным группам, и наоборот, при снятом флажке линейно, с сортировкой по коду товара в весах.

       Сортировать по коду товара внутри группы - опция для удобства запоминая кодов товара персоналом.

    Выводить отчет в два столбца - для экономии бумаги при распечатке кодов на весы для персонала.

    Не отбирать скрытые товары - при заполнении списка товаров на программирование в весы не будут заноситься товары с пометкой "скрытый" (см. раздел "Справочник товаров").

    Программировать название магазина: - разрешение печати на штрихкодовой этикетке, получаемой с весов, названия магазина или предприятия. Укажите в строке под данной галочкой.

    Печать названия товара в 2 строки для 58х40 - зависит от тех.возможностей весов. Дает больше информативности.

    Разрешить программирование кнопок прямого доступа весов самообслуживания BS - делает активным соответствующее меню и форму в Оборудование: Весы с печатью. Применяется для программирования весов самообслуживания с большой клавиатурой на стойке, где клиенты самостоятельно взвешивают/печатают этикетку нажимая на крупные кнопки с изображениями товара.

    Временной интервал в днях... - задается интервал чистки весового "мусора" для разблокирования памяти весов от старых неиспользуемых весовых кодов. Значение по умолчанию: 365.
