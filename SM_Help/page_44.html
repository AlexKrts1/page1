    Данная возможность служит для отбора списка товаров товаров по товарной группе, а для расширеных режимов отображения и по другим характеристикам.
    Значения параметров отбора понятны из их наименований.
    Красным кружком обведен флажок множественного выбора торговых марок. При установленном флажке, нажатие на троеточие в окошке выбора торговых марок откроет их справочник, где воспользуйтесь опцией множественного выбора в главном меню справочника (см.соответствующий раздел справки).
