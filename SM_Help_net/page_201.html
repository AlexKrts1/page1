Другие отчеты

    Отчеты, не попадающие ни в одну из рассмотренных ранее категорий, будут описаны в данном подразделе. На рисунке части главного меню, ниже, они выделены зелеными прямоугольниками.
    1. Товары, которые не продавались после конечной даты (кроме ***). Формирует выборку для анализа товарного справочника и/или для поиска ошибок, задвоенных кодов и т.д. (подробнее еще см.раздел справки "Поиск ошибок"). Для выборки использует значение календарика "По дату" и значение товарной группы, выбранные на форме. В отбор не попадают товары в наименовании которых есть *** (три звездочки, таким образом некоторые клиенты помечают зарезервированные товарные позиции в справочнике товаров).     2. Отчет "Книга продаж/возвратов". Выводит отчет в несколько усовершенствованной форме отчетности "Книги продаж/возвратов" (обязательна для ведения частными предпринимателями на упрощенной системе налогообложения). По сравнению с исходным вариантом добавлено развертывание по товарам, т.к. каждый товар в чеке может иметь своего поставщика, соответственно и номер приходной накладной нашей и расходной накладной поставщика для каждого товара будут разными.  Внимание! Для полной корректости данных необходимо снимать отчет ежедневно после окончания рабочей смены. Пример выборки - на рисунке ниже:
    3. Экспорт. Выгрузки в текстовые файлы разных документов (не только реализации) для взаимодействия с разным дополнительным учетным ПО. Разделитель - табуляция. По поводу согласования других форматов экспорта итоговых данных обращайтесь к разработчику ПО "SuperMarket".
