Отчеты ОПС

    Специализированные отчеты для сети магазинов потребительской кооперации "КоопМаркет" позволяют формировать отчеты для отправки отчетности в вышестоящие структуры кооперации, для сводного анализа данных с магазинов всей сети по стране.
    Для выборки лишь требуется указать отчетный период (обычно месяц), и выбрать в главном меню тип отчета.
     
    В открывшемся отчете нажать на значок принтера и внизу появившейся формы печати выбрать PrintToFile, указать требуемый на текущий момент сетью кооперации формат сохраняемого файла, указать название файла и куда его сохранить, нажать ОК.
    Дальше останется только переслать сохраненные файлы в вышестоящий статистический отдел.
