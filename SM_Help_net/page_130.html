Журнал дисконтных скидок

    Этот журнал позволяет хранить, просматривать сделанные по чекам и расходным накладным скидки. Для информации по самим скидкам смотрите разделы справки: "Настройки - вкладка Скидки и Скидки 2", "Чек".
    По двойному клику мышкой на интересующей вас строке откроется чек, по которому была предоставлена данная скидка.
    Меню печать в двух вариантах предлагает предварительный просмотр и печать реестра скидок, например, вот как выглядит второй вариант:
    Пункт главного меню "Статус формы оплаты чека" позволяет изменить форму оплаты в записи о скидке и в самом чеке.
    Эта функция иногда необходима в процессе исправления ошибок кассиров.
    И конечно, есть фильтр данного журнала, позволяющий отобрать записи согласно требуемых критериев отбора.
    Значения опций отбора понятны из их названий. Рассмотрим подробнее всплывающее меню кнопки "Печать реестра".
    В данном меню доступны такие готовые отчеты и следующие критерии отбора в них используются:
    1. Реестр чеков по дате/владельцу - используются с даты/по дату, Владелец (если указано "Все владельцы", то выводится общий отчет, иначе - по выбранному), вывод в разрезе сумм по каждому чеку, потом итого по карте;
    2. Реестр сводный по всем владельцам - используются только даты, вывод в разрезе сумм по каждой карте, чеки объединены;
    3. Отчет-анализ по дисконту с реквизитами - используются только даты, вывод в разрезе только суммы по каждой карте, без расшифровки наименований, плюс дополнительные данные по владельцам (адрес, телефоны, электронная почта);
    4. Отчет по погашениям бартера клиентами - используются только даты, клиенты с нулевым сальдо не отбираются.
