Нефискальный принтер


    Использовать принтер нефискальных чеков - активация возможности. Настройки справа окна демонстрируют характеристики связи, их можно изменять.

    Заголовок чека должен состоять из 4х строк если флажок "Печатать № чека" не установлен или из 3х, если установлен. Примечание: в конце текста не должно быть лишних строк и пробелов. Возможно для вашего принтера потребуется подобрать требуемое количество строк, для достижения ровной обрезки ленты. Задача обычно заключается в том, чтобы печатать заголовок чека препринтом, т.е. заранее при печати предыдущего чека - так достигается максимальная экономия бумажной ленты.

    Сообщение в конце чека - одна строка. Примечание: в конце текста не должно быть лишних строк и пробелов. Возможно для вашего принтера потребуется подобрать требуемое количество строк, для достижения ровной обрезки ленты. Задача обычно заключается в том, чтобы печатать заголовок чека препринтом, т.е. заранее при печати предыдущего чека - так достигается максимальная экономия бумажной ленты.

    Печать скидки как бонуса - чек печатается как будто без скидки, обрезается, затем печатается "бонусный чек" в котором указана начисленная скидка, снова обрезается. Часто используется строительными магазинами для прорабов - они отчитываются по первому чеку.

    Печатать дату / время на чеке, Печатать имя кассира в чеке, Печатать номер чека - значения опций ясны из их названий.

    Печатать код товара в чеке - значения опций ясны из их названий.

    Печатать название склада - значения опций ясны из их названий.

    Печатать имя кассира в чеке - значения опций ясны из их названий.

    Печатать и фискальные позиции - товары могут иметь ряд признаков разделяющих их по типам печати в чеках/принтерах. Данный флажок позволяет печатать как нефискальные, так и фискальные позиции, занесенные в чек. При правильном использовании служит для корректировки ошибок ввода товаров в базу данных (подробнее см. раздел "Справочник товаров").

    Печатать № чека  - значения опций ясны из их названий.

    Автообрезка чеков - после печати чека компьютер посылает команду о частичной обрезке чека на принтер.

    Печатать номер кассы - значения опций ясны из их названий.

    Печатать тип оплаты чека - значения опций ясны из их названий.

    Печатать сумму по нефискальным позициям отдельно - для более удобного понимания сколько денег в сводном чеке нужно положить в нефискальную кассу.

    По окончании чека открывать денежный ящик - для большего удобства и контроля кассиров. Разъем денежного ящика RJ-25 (RJ-12) должен быть обжат на контакты 4 и 5.

    Момент открытия ящика - выберите вариант или ящик откроется уже после печати чека или сразу при открытии формы расчета сдачи (сразу при нажатии кассиром сочетания клавиш Ctrl+F12 или других для сохранения чека).  

    Убирать двойной интервал между строками - для некоторых моделей принтеров, если они печатают "через строку" может понадобиться эта опция.
