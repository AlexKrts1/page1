Журнал переоценки

    Этот журнал позволяет работать с сохраненными ранее документами переоценки товаров, как созданными автоматически при проведении документов, так и ручными переоценками.
    Окно журнала состоит из двух частей: в верхнем отображаются заголовки переоценок, в нижнем - содержимое переоценки, заголовок которой выделен в верхней части окна.Если вы переоценили товар ошибочно, то по умолчанию доступна только одна возможность откатить назад изменения - "Распровести одну строку". При этом появится следующий вопрос:
    Чтобы распровести все переоценку сразу или удалить строку без распроводки есть соответствующие служебные опции в настройках, однако необдуманные действия по особенно удалению без распроводки строк могут привести к неправильному подсчету товарно-денежного отчета!
    Предусмотрено несколько вариантов печати переоценок:
    1. Печать списка-реестра - выводится на печать список заголовков отобранных переоценок с подсчетом итогов;
    2. Печать переоценки - печатается только текущая переоценка в развернутом виде. Пример показан на рисунке ниже:
    Фильтр/результаты - позволяет отобрать документы с задаными параметрами:
     
    Значения критериев отбора понятны из их названий.
    Внизу окна фильтра есть кнопка "Печать реестра". При нажатии появляется всплывающее меню с дополнительными отчетами:
    Сводная ведомость за период - объединяет все переучеты, которые попадают в диапазон дат, указанных вверху фильтра, в единый бланк. 
    Список партий и номеров документов - выводит на печать список строк переоценок, которые отвечают следующим критериям:
       1. Обязательно дата в промежутке диапазона дат указанных вверху формы фильтра;
       2. Если указано Примечание, то к условию поиска прибавится поиск в примечании;
       3. Если указан номер приходной накладной, то к условию поиска прибавится отбор по этому номеру в поле номера родительской приходной накладной связанной партии.
       4. Если указан номер внутренней накладной, то к условию поиска прибавится отбор по этому номеру в поле номера родительской внутренней накладной связанной партии.
