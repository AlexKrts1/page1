Создание калькуляции

    Заголовок окна:
    Номер - присваивается автоматически.
    Название калькуляции - до 55 символов.
    Получаемый товар штучный - если установить, то это означает, что нужно указывать такое количество ингредиентов (суммарно по нетто), которое будет израсходовано для приготовления 1шт. производимого товара.
    Порция в готовом виде, кг - если блюдо будет продаваться наразвес по цене за 1кг, тогда укажите вес по нетто закладки на которую подобраны ингредиенты; если же получаемый товар штучный, то укажите вес нетто порции блюда, при этом суммарный вес нетто ингредиентов должен быть равен этому весу порции или должен делиться нацело на вес порции (например нетто всех ингредиентов салата составлено на 1 кг, вес порции 0,200кг, при составлении акта производства на 2 закладки будет произведено 10 порций готового салата  (1/0,2)*2=10  ).
    Продажная цена блюда (по нетто), грн - автоматически рассчитывается исходя из розничных цен, указанных в теле калькуляции, делением суммы стоимости нетто по калькуляции на количество порций в одной закладке.
    Первая вкладка "Добавление / редактирование калькуляции" - "Общие данные и рецептура".
    Тип калькуляции - "Производство" по умолчанию.
    Производимый товар - выберите из справочника товаров готовый или создайте код товара, под которым товар будет продаваться.
    Скрытая калькуляция - опция для скрытия в соответствующем режиме отображения неиспользуемых калькуляций в справочнике калькуляций.
    Рекомендуемая розница, грн - подсчитываемая автоматически по факту сохранения калькуляции розничная стоимость 1 единицы производимого блюда (штучной или весовой, в зависимости от выбранных опций).
    Пользователь - автоматически присваивается автор по тому, кто вошел в программу.
    Рецептура - произвольное описание процесса приготовления блюда. Необязательный параметр.
    *
      Вторая вкладка "Добавление / редактирование калькуляции" - "Ингредиенты". Пример весового салата, закладка на 1 кг, порция производства 1кг, цена за 1кг.
      Пример порционного салата, закладка на 1 кг, порция производства 0,200кг, цена за порцию. Каждая 1 закладка в Акте производства будет производить 5шт порций салата.
      Для добавления/редактирования ингредиентов в нижней части окна есть контекстное меню.
      В этом окне для составления калькуляций для Актов производства указывайте нормы количества ингредиентов, необходимых для получения одной закладки готового изделия.
      При нажатии "Добавить" контекстного меню, вначале появляется справочник товаров, в котором для удобства подбора можно наложить фильтр по товарной группе №999 "Ингредиенты". После выбора ингредиента (в данном случае при помощи двойного клика мышкой по нужной позиции) появляется диалоговое окно, при помощи которого вводятся данные о количестве брутто/нетто, ценах закупочных/розничных каждого ингредиента. Два нижних параметра предназначены для Актов разделки и в калькуляциях для Актов производства не используются.
      Готовую калькуляцию после сохранения можно распечатать.
      Для дальнейших сведений о Актах производства, смотрите соответствующий раздел справки.
