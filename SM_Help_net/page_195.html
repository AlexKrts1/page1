Вкладка Результат

    Третья вкладка дизайнера используется для работы с возвращенными запросом SELECT данными.
    Панель инструментов обеспечивает функционал для работы с полученными данными.
    Если данные, в полученном виде, вам нужны для анализа в Текстовом файле или Excel, то соответствующая кнопка позволит экспортировать их в txt или xls файл, по умолчанию находящийся в одном каталоге с программой.
    Обновить запрос - снова выполняет текст запроса и таким образом перечитывает данные с сервера/базы данных.
    Очистить таблицу - освобождает память, очищая результирующую таблицу.
    Количество записей - подсчитывается автоматически, выводится в информационных целях.
    Макетировать отчет - открывает специальную редактируемую печатную форму для создания, изменения и сохранения результата запроса в виде отчета, выводимого на принтер. Смотрите следующий подраздел справки для более подробной информации.
