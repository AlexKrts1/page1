Отчеты по чекам

    Отчеты по чекам с готовыми печатными формами, отличающимися от общих печатных форм, при выборке по параметрам формы, выделены в отдельные пункты главного меню. В главном меню, на рисунке внизу, они выделены синими прямоугольниками. Все они используют для выборки значения календариков "С даты/По дату", а некоторые из них и другие параметры, выбираемые на форме отчета, о чем будет указано в описании ниже.
    1. Отчет-анализ продаж в разрезе поставщиков. С формы используются даты, склад, товарная группа, опция по неучету корректировочных чеков.  Итоги по каждому поставщику и затем по всему отчету. Пример на рисунке ниже:
    2. Отчет продаж за период суммарный краткий. Используются даты и склад. Вывод в два столбца. Цены розничные усредненные, кол-во общее. Группировка по товарным группам. Итоги по каждой группе и затем по всему отчету.
    3. Отчет продаж за период подробный с количеством продаж. Используются даты и склад. Цены усреднены, остаток суммарный. Группировка по товарным группам. Итоги по каждой группе и затем по всему отчету. Содержит подсчитываемое в целях маркетингового анализа количество событий продаж каждого товара.     4. Отчет реализации в разрезе кассовых мест. Подробный отчет по товарам, сгруппированный по кассовым местам. Используются только даты. Корректировочные чеки не учитываются. Выводятся закупочная и розничная суммы, наценка усредняется. Итоги по каждому кассовому месту, затем по всему отчету.
    5. Отчет приход/реализация развернутый по чекам. Интересный отчет, который с формы выбора параметров использует только выбранный товар. Далее по этому товару отбираются все приходные накладные, а по каждой накладной отображается почековая реализация данного товара. Даты не используются, отчет формируется за весь период работы магазина. Данный отчет обычно используется при проведении расследований почему остаток товара именно такой как отображается программой, а также при поиске злоупотреблений и ошибок персонала.
    6. Отчет-анализ чеков по товарным группам и внутреннему контрагенту. Использует даты и контрагента, выбранных на форме. Отобранный на форме контрагент означает  внутреннего контрагента (внутреннюю фирму), на которого был оприходован отгружаемый товар. Если магазин ведет учет по нескольким внутренним контрагентам, то данный отчет может оказаться весьма полезным. Дизайн отчета аналогичен отчетам №1, №2, №5 предыдущего подраздела "Отчеты по расходным накладным".
    7. Отчет-анализ 1склад фиск/н.ф. + 2 склад остаток. Используются только даты, корректировочные чеки не учитываются. Группировка товаров по товарным группам. Итоги по каждой группе и по всему отчету. Результат отчета: по каждому товару по складу №1 - кол-во проданного в фиск.чеках (статус печати чека "Да") и кол-во проданного в нефискальных чеках (статус отличен от "Да"), партионный остаток по складу №2. Отчет был создан по конкретному тех.заданию под заказчика, возможно малопригоден в широком использовании.
    8. Отчет-анализ продаж в разрезе отделов и товарных/групп. Используются даты. Группировка реализации товаров по отделам, по товарным группам. Итоги по отделам, товарным группам, по всему отчету. Наценка усреднена, количество суммарное по коду товара. Пример дизайна отчета как в следующем пункте №9.
    9. Отчет-анализ продаж в разрезе одного отдела. Все аналогично предыдущему №8, только при запуске отчета еще будет выведен справочник отделов, где двойным кликом мышки нужно выбрать интересующий вас отдел.
